**Installation**

How to install
```
clone the repo
run python3 database_creations.py
run python3 api_main.py
open web browser and type in the URL: 127.0.0.1/index
```
**Usage**

- This program is used to successfully store and manage passwords; it combines safety and organizational concerns into one program.

**Roadmap**

Our future development goals for this project include:

- **Enhanced Security Features**  
    Implement multi-factor authentication, stronger encryption options, and improved password-strength analysis.
- **User Interface Improvements**  
    Develop a more modern, intuitive frontend and refine navigation for smoother user experience.
- **Database Expansion**  
    Add support for additional database systems and improve backup/restore options for stored passwords.
- **Cross-Platform Compatibility**  
    Expand compatibility for macOS, Linux, and Windows, along with potential mobile support.
- **Automated Password Generation**  
    Integrate a customizable password generator to streamline creating secure passwords.
- **Account Management Features**  
    Add user roles, account recovery mechanisms, and profile customization.
- **Cloud Sync Support**  
    Optional cloud-based storage enabling users to securely access their vault across multiple devices.
